# 15 January 2026

This is the initial release of carrion-strap. This version focused on profiles and units.

Carrion-strap released with the following:

- Text:
  - Text formatting
  - Font
  - Text alignment
- Advanced utilities:
  - Blocks
  - Details-related classes
  - Display, flex, and spacing
- Components:
  - Gallery
  - Profile
  - Censor
  - Static
- Extra modules:
  - Units

Chat theming is still a work in progress.
